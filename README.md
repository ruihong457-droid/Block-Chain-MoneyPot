# Block-Chain-MoneyPot
MoenyPot, A decentralized finance (DeFi) application built on blockchain technology that reimagines traditional group savings funds.

## Project Overview
An Ethereum-based multi-person savings contract that solves trust issues when friends save money together. Implements multi-signature approval through smart contracts to ensure secure and transparent fund management.

## Core Features
- Create savings pots (set approvers and minimum approvals)
- Multi-person deposit functionality
- Withdrawal approval process (m-of-n multi-signature)
- Real-time status tracking

## Tech Stack
- Smart Contracts: Solidity, Ethereum Sepolia
- Frontend: HTML/CSS/JavaScript, Ethers.js v6
- Wallet: MetaMask

## Quick Start
1. Deploy MoneyPot.sol contract in Remix IDE
2. Configure contract address in frontend code
3. Run local server to access the application

## Usage Flow
Connect Wallet → Create Pot → Deposit → Create Withdrawal Request → Approve → Execute Withdrawal
